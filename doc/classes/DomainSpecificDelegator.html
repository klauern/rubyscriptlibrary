Class DomainSpecificDelegator
In: lib/DomainSpecificLanguage.rb
Parent: DomainSpecificLanguage

A DomainSpecificDelegator is a DSL that delegates methods to a class or object. The main use is to separate the mechanics of scoping from the methods of a utility class.

Methods

delegate_to   method_missing  

Public Class methods

delegate_to(receiver, method_hash)

insert one or more delegate_to calls in the class definition, giving a receiver and a hash. Each hash pair is of the form verb => method where verb is the name you will use in the DSL and method is the method in the receiver that will handle it.
