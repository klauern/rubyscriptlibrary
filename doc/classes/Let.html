Class Let
In: lib/DomainSpecificLanguage.rb
Parent: DomainSpecificLanguage

Let is a DomainSpecificLanguage that actually creates DomainSpecificLanguages.

Let works a lot like let in Scheme. Your provide a hash of names and value expressions. The value expressions are evaluated in the outer scope, and then we have a little domain specific language wher ethe inner scope has the names all bound to the values. E.g. <tt> with Let do 

    let :x => 100, :y => 50 do
        print "#{x + y} should equal fifty"
    end

end </tt>

Within the Let DomainSpecificLanguage, you can declare multiple let statements and nest them as you please.

One important limitation: you cannot bind a value to a name that is already a local variable.
